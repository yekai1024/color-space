# Color Space

[English](README.md) | [中文](README.zh-CN.md)

**✨ Color Space** 是一款兼容 VS Code 的扩展插件，能够为您的每个工作区自动分配独特的颜色。这可以帮助您一目了然地快速区分不同的项目窗口。

![color-theme](images/color-theme.png)

### 功能特性
- 🎨 **自动调色**: 根据工作区自动选择合适的配置，让你惊喜。
- 🎨 **创意模式**: 使用全新的交互式色盘，实时预览并创建您专属的主题配色。
- 🏛️ **主题宝库**: 预设多种优雅配色，莫兰迪深色和浅色主题，以及中国敦煌主题色，总有一款适合你！
- 🖱️ **状态栏集成**: 在状态栏轻轻一点“调色盘”，即可秒速切换不同主题。
- 🎲 **手动控制**: 想玩点花样？手动选择或使用`惊喜`模式，即可变换工作区颜色。
- ⚙️ **自由配置**: 全局开关随心设，还能把“不想打扮”的工作区拉进忽略名单，彻底放权给你。


### 命令

- `Color Space: Enable` - 启用自动设置工作区主题。
- `Color Space: Disable` - 禁用自动设置工作区主题。
- `Color Space: Color Theme` - 显示系统预置的颜色主题。
- `Color Space: Pick Color` - 从主题列表中快速选择。
- `Color Space: Surprise Me` - 随机选择主题。
- `Color Space: Clear Color` - 清除当前工作区主题颜色。

### 扩展设置

本插件提供以下设置：

* `colorspace.enabled`: 启用/禁用插件。
* `colorspace.ignoreList`: 忽略的工作区名称列表。
